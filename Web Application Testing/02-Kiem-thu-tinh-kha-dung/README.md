# 02. Kiểm thử tính khả dụng

## Mục tiêu

Đánh giá mức độ dễ dùng, dễ hiểu và nhất quán của giao diện đối với dược sĩ, nhân viên bán hàng, quản lý sản phẩm và Admin.

## Checklist

| Mã | Nội dung kiểm thử | Kết quả |
|---|---|---|
| USA-01 | Menu chính dễ nhìn, rõ vai trò | Pass |
| USA-02 | Các nút thêm/sửa/xóa/huỷ có vị trí dễ nhận biết | Pass |
| USA-03 | Form có placeholder và nhãn nhập liệu | Pass |
| USA-04 | Thông báo lỗi khi nhập dữ liệu không hợp lệ | Pass |
| USA-05 | Tài liệu hướng dẫn cài đặt và sử dụng có trong `docs` | Pass |
| USA-06 | README có hướng dẫn chạy nhanh, tài khoản demo và ảnh minh họa | Pass |
| USA-07 | Chatbot có thể hướng dẫn người dùng theo tài liệu nội bộ | Pass |
| USA-08 | Người dùng mới cần biết vào `/admin-login` cho Admin | Pass, đã ghi trong README |

## Nhận xét

Giao diện có đủ menu, dashboard, bảng dữ liệu và form nghiệp vụ. README, PDF hướng dẫn, screenshots và chatbot giúp tăng tính dễ dùng khi bàn giao cho nhóm hoặc giảng viên.

## Khuyến nghị

- Bổ sung breadcrumb cho các trang sâu nếu muốn tăng khả năng định vị.
- Thêm hướng dẫn nhỏ ngay trong màn hình tạo hóa đơn/phiếu nhập nếu người dùng mới thao tác lần đầu.
